Schedule and Publish TikTok Videos

Nalyse is a web-based tool that helps users schedule and publish videos to a TikTok account they own using TikTok’s official Content Posting API.

Connect TikTok Account

You will be redirected to TikTok to authorize posting access, then returned to a confirmation page.

Reviewer Instructions

  1. Click Connect TikTok Account
  2. Sign in to TikTok and approve permissions
  3. You will be redirected to a confirmation page indicating the account is connected

How it works

  • Connect your TikTok account securely using TikTok OAuth (Login Kit)
  • Choose a video, caption, and scheduled publish time
  • Publish automatically using TikTok’s Content Posting API

Compliance

Nalyse only publishes content explicitly authorized by the user. It does not scrape data, automate engagement, or interact with accounts the user does not own.
